# Customer Churn Prediction

## Problem Statement
Predict whether a customer will churn using customer demographic and service usage data.

## Tools Used
Python, Pandas, NumPy, Seaborn, Scikit-learn

## Models Used
- Logistic Regression
- Random Forest

## Evaluation Metrics
Recall, Precision, F1-score, ROC-AUC

## Key Insights
- Contract type and tenure strongly affect churn
- High monthly charges lead to higher churn

## Business Impact
Helps businesses identify high-risk customers and reduce churn through targeted actions.
